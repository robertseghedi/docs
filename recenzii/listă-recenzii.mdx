## Funcționalități

### Schimbă status

Dă click dreapta pe recenzie și du-te la opțiunea **schimbă status**. Acolo vei găsi următoarele patru variante: aprobate, respinse, în așteptare și în arhivă. Pentru a modifica statusul recenziei, doar selectați una din variantele menționate.

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### Editează

Dă click dreapta pe recenzie și selectează opțiunea "Editează". În funcție de ce îți dorești să faci, ai următoarele variante:

**Corectează:** dacă vrei să corectezi recenzia, o poți face cu AI prin butonul "Corectează" sau poți modifica manual selectând căsuța cu textul recenziei și modificând ce dorești.

**îmbunătățește:** poți îmbunătăți recenzia cu AI prin butonul "îmbunătățește", acesta oferindu-vă o alternativă a recenziei, ce poate fi acceptată și modificată instant.

**Sugestii AI:** în cazul în care simți că ceva poate fi făcut mai bine la recenzie, dar nu știi ce, prin butonul "Sugestii AI" vei primi câteva sugestii de îmbunătățire, pe care le poți implementa ulterior singur.

**Evidențiează:** Pentru evidențierea unor cuvinte, în lista cu toate recenziile dai click pe cuvintele pe care le vrei evidențiate. Acestea își vor schimba culoarea și atunci vei știi că au fost evidențiate. 

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### Filtrează căutările

Pentru o filtrare amănunțită a căutărilor ai două căsuțe:

**Statusul recenziilor:** ai varianta de a căuta doar recenzii cu un anumit status, selectându-l. 

**Nume sau recenzie:** poți alege să cauți doar după una din aceste două variante, programul blocându-vă cealaltă variantă atunci când una este selectată.

În funcție de cât de amănunțită dorești să fie filtrarea căutării, poți alege din cele două căsuțe sau poți selecta varianta toate la amândouă.

Odată ce te-ai hotărât cât de amănunțit vrei să cauți, te duci la căsuța "Caută o recenzie" și scrii informația după care vrei să cauți recenzia.

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```
