## Funcționalități

### Pagina de formular

În pagina de formular ai următoarele opțiuni:

**Modificarea formularului:** vei avea deja un text prestabilit, dar dacă îți dorești să-l modifici, o poți face foarte simplu. Doar apeși pe text și începi să-l modifici.

**Activarea link-ului de formular:** pentru a îți activa sau dezactiva formularul ai butonul "Live". Dacă apeși pe el ai buton pentru activare/dezactivare și un preview al link-ului, pe care îl poți copia apăsând pe iconița din dreapta lui.

**Design:** dacă vrei să modifici tematica paginii, în dreapta butonului "Live" este butonul "Design". Apăsând pe el ți se va deschide o listă cu mai multe tematici cu care poți schimba tematica actuală.

**Refresh:** pentru a vedea cum se comportă pagina fără a intra pe link, poți da refresh prin butonul din colțul din dreapta, sus.

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### Wall of love

Use [fenced code blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) by enclosing code in three backticks and follow the leading ticks with the programming language of your snippet to get syntax highlighting. Optionally, you can also write the name of your code after the programming language.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
